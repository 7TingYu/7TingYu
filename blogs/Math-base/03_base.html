# 考情分析

本章历年考核分值 14-18 分，题型以选择题、简单计算和计算题为主，本章重点为：导 数的概念及其几何意义和作为变化率的实际意义，各种求导法则和基本初等函数的导数和微 分公式；重点掌握复合函数的求导法则，隐函数求导法.本章常见考试题型：

  1. 导数的几何意义.
  2. 讨论分段函数分段点的连续性与可导性.
  3. 求函数的导数：复合函数求导， 隐含数求导，参数方程求导
