a1 / a2 != b1 / b2 , 有唯一解，两直线相交

# 代数不等式（低频且易）【01010103】

1. 不等式 b > a 与 a - b > 0 等价，常用不等式性质：
    若 a > b ， c > b ，则 a > c
    若 a > b ， k > 0 ，则 ka > kb 
    若 a > b ， k < 0 ，则 ka < kb 
    若 a > b ， d > c ，则 a + c > b + d , a - d > b - c
2. 常见基本不等式：
    a, b 为非负实数, (a + b) / 2 >= (ab) ^ 1/ 2
    a, b 为任意实数, |a + b| <= |a| + |b|
